隱私政策

生效日期：2026年1月9日

1. 前言

ETOS LLM Studio 非常重視您的隱私。本應用採用 「本地優先 (Local-first)」 的設計理念。我們不運營任何中央伺服器，也不會存儲您的聊天記錄、語音錄音或任何個人信息。

核心原則：您的數據完全由您掌控。

2. 數據處理說明

  • 本地存儲： 所有的聊天會話、配置參數和記憶碎片都安全地存儲在您設備的 App 沙盒中。
  • AI 處理： 您發送的數據（文本、音頻、圖片）將直接發送給您自行配置的 第三方 AI 服務提供商（如 OpenAI, Anthropic, Cloudflare 或自建模型）。我們不會攔截、處理或存儲這些流量。
  • 語音與音頻： 麥克風權限僅在您主動觸發錄音時啟用。錄音文件僅用於發送至您配置的 API 進行轉寫或分析。
  • 多端同步： iPhone 與 Apple Watch 之間的數據同步通過 WatchConnectivity 框架完成。這是一種點對點的直接加密傳輸，確保同步過程中數據不會離開您的個人設備生態系統。
  • 本地網絡： 僅用於可選的「本地調試」功能，以便您通過局域網內的瀏覽器管理 App。

3. 透明度承諾

ETOS LLM Studio 不包含 任何行為追蹤代碼，也不會收集任何分析數據。作為一款極客工具，我們追求的是完全的透明度和用戶對大模型工作流的絕對所有權。
