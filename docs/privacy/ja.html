プライバシーポリシー

発効日：2026年1月9日

1. はじめに

ETOS LLM Studio は、お客様のプライバシーを非常に重視しています。本アプリは「ローカルファースト (Local-first)」の設計理念を採用しています。中央サーバーは運用しておらず、チャットログ、音声録音、個人情報を保存することはありません。

基本原則：データは完全にあなたのコントロール下にあります。

2. データ処理について

  • ローカルストレージ： すべてのチャットセッション、設定パラメータ、記憶の断片は、デバイス内の App Sandbox に安全に保存されます。
  • AI 処理： 送信したデータ（テキスト、音声、画像）は、お客様自身が設定したサードパーティ AI サービスプロバイダー（例：OpenAI, Anthropic, Cloudflare または自作モデル）へ直接送信されます。私たちはこれらのトラフィックを傍受、処理、保存することはありません。
  • 音声とオーディオ： マイクへのアクセス権限は、お客様が能動的に録音を開始したときのみ有効になります。録音ファイルは、書き起こしや分析のために設定された API へ送信する目的でのみ使用されます。
  • マルチデバイス同期： iPhone と Apple Watch 間のデータ同期は、WatchConnectivity フレームワークを通じて行われます。これは直接的な暗号化されたピアツーピア通信であり、同期プロセス中にデータが個人のデバイスエコシステムの外に出ることはありません。
  • ローカルネットワーク： 任意の「ローカルデバッグ」機能でのみ使用され、ローカルネットワーク内のブラウザ経由でアプリ内のファイルを管理するために使用されます。

3. 透明性へのコミットメント

ETOS LLM Studio には行動追跡コードは含まれておらず、分析データも収集しません。ギーク向けのツールとして、私たちは完全な透明性と、LLM ワークフローに対するユーザーの絶対的な所有権を追求しています。

4. 免責事項

  • 中立的なツール属性： 本アプリは汎用的な LLM クライアントツールとしてのみ機能します。アプリのコードには、特定の商用 API サービスのエンドポイントは事前設定されていません。
  • コンテンツの責任： すべての AI 生成コンテンツ（テキスト、画像、音声を含む）は、お客様自身が設定したサードパーティのサービスプロバイダーによって生成されます。私たちはそのコンテンツを管理、介入、または審査することはできないため、生成されたコンテンツの正確性、合法性、またはコンプライアンスについて一切の責任を負いません。
  • サービスの選択： お客様が使用することを選択したすべての API サービス（ドキュメントで推奨しているものを含む）は、お客様個人の選択によるものです。これらのサービスの利用に伴う費用およびリスクは、お客様自身の負担となります。
