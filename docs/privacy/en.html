Privacy Policy

Effective Date: January 23, 2026

1. Introduction

ETOS LLM Studio takes your privacy very seriously. This application adopts a "Local-first" design philosophy. We never store your chat logs, voice recordings, or any personal identity information.

Core Principle: Your sensitive data is completely under your control.

2. Data Processing

  • Local Storage: All chat sessions, configuration parameters, and memory fragments are securely stored within the App Sandbox on your device.
  • AI Processing: The data you send (text, audio, images) is sent directly to the third-party AI service providers you configure yourself (e.g., OpenAI, Anthropic, Cloudflare, or self-hosted models). We do not intercept, process, or store this traffic.
  • Voice & Audio: Microphone permission is only enabled when you actively trigger recording. Audio files are only used to be sent to your configured API for transcription or analysis.
  • Multi-device Sync: Data synchronization between iPhone and Apple Watch is done via the WatchConnectivity framework. This is a direct, encrypted peer-to-peer transmission, ensuring that data never leaves your personal device ecosystem during sync.
  • Local Network: Used only for the optional "Remote File Access" feature, allowing you to manage App files and configurations via specialized tools on your local network.

3. Transparency Commitment

ETOS LLM Studio promises not to track your specific behavioral paths and never reads your chat content. We only collect minimal, anonymous activity status data (e.g., daily active users, version distribution) to evaluate project health.

4. Disclaimer

  • Neutral Tool Attribute: This application serves only as a generic LLM client tool. The application may pre-configure some commercial API service endpoints for convenience, but by default, it **does not include** any valid API keys or access credentials.
  • Content Liability: All AI-generated content (including text, images, and audio) is generated by the third-party service providers you configure yourself. We cannot control, intervene in, or review their content, and therefore assume no responsibility for the accuracy, legality, or compliance of the generated content.
  • Service Selection: Any API service you choose to use (including those we may recommend in documentation) is your personal choice. You assume all costs and risks associated with using these services.
