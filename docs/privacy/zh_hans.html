隐私政策

生效日期：2026年1月9日

1. 前言

ETOS LLM Studio 非常重视您的隐私。本应用采用 “本地优先 (Local-first)” 的设计理念。我们不运营任何中央服务器，也不会存储您的聊天记录、语音录音或任何个人信息。

核心原则：您的数据完全由您掌控。

2. 数据处理说明

  • 本地存储： 所有的聊天会话、配置参数和记忆碎片都安全地存储在您设备的 App 沙盒中。
  • AI 处理： 您发送的数据（文本、音频、图片）将直接发送给您自行配置的 第三方 AI 服务提供商（如 OpenAI, Anthropic, Cloudflare 或自建模型）。我们不会拦截、处理或存储这些流量。
  • 语音与音频： 麦克风权限仅在您主动触发录音时启用。录音文件仅用于发送至您配置的 API 进行转写或分析。
  • 多端同步： iPhone 与 Apple Watch 之间的数据同步通过 WatchConnectivity 框架完成。这是一种点对点的直接加密传输，确保同步过程中数据不会离开您的个人设备生态系统。
  • 本地网络： 仅用于可选的“本地调试”功能，以便您通过局域网内的浏览器管理 App。

3. 透明度承诺

ETOS LLM Studio 不包含 任何行为追踪代码，也不会收集任何分析数据。作为一款极客工具，我们追求的是完全的透明度和用户对大模型工作流的绝对所有权。

4. 免责声明

  • 中立工具属性： 本应用仅作为通用的 LLM 客户端工具。应用代码中不预设任何特定的商业 API 服务端点。
  • 内容责任： 所有 AI 生成的内容（包括文本、图片、音频）均由您自行配置的第三方服务提供商生成。我们无法控制、干预或审查其内容，因此不对生成内容的准确性、合法性或合规性承担任何责任。
  • 服务选择： 您选择使用的任何 API 服务（包括我们可能在文档中推荐的服务）均属于您的个人选择。您需自行承担使用这些服务产生的费用及风险。
