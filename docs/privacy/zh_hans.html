隐私政策

生效日期：2026年3月22日

1. 前言

ETOS LLM Studio 采用 “本地优先（Local-first）” 的设计思路。大多数核心数据保存在您的设备本地，由您自己掌控。

我们不会在开发者控制的服务器上默认存储您的聊天内容、语音录音、图片附件或 API Key；但应用本身确实会在设备本地保存聊天记录、配置、记忆、请求日志等数据，以支撑同步、统计、Daily Pulse 和年度汇总等功能。

2. 设备本地存储与本地处理

  • 聊天与会话数据：聊天记录、会话配置、附件引用、世界书、记忆、Daily Pulse 数据、反馈历史和应用设置会保存在应用沙盒内。
  • API Key：API Key 通过系统 Keychain 保存，而不是写入普通 JSON 配置文件；如果您启用了 Apple 的相关能力，它也可能通过 iCloud 钥匙串在您自己的设备之间同步。
  • 本地请求日志：应用会在本地记录提供商名称、模型标识、请求时间、完成时间、状态、是否流式、Token 用量等元数据。这些日志不包含聊天正文，主要用于流量分析、响应测速、Daily Pulse 统计和年度汇总等本地功能。
  • 反馈工单缓存：如果您使用反馈助手，应用会在本地保存工单编号、票据令牌、状态、更新时间和公开链接，方便后续追踪处理进度。

3. AI 请求、音频与图片

  • 文本聊天：您的文本消息会直接发送到您自行配置的第三方 AI 提供商（如 OpenAI、Anthropic、Google、兼容 OpenAI 的服务或自建模型）。我们不会在开发者服务器上拦截或转存这些内容。
  • 语音与音频：麦克风权限仅在您主动录音或使用语音输入时启用。录音只会发送给当前模型提供商或您主动选择的语音/转写服务。
  • 图片与文件：应用仅在您主动选择时访问图片或文件。相关附件会发送到当前模型提供商，或发送到您主动连接的本地调试端。

4. 同步、本地网络与 Apple 服务

  • iPhone / Apple Watch 同步：两端实时同步通过 WatchConnectivity 完成，不经过开发者自建服务器。
  • iCloud 同步：如果您主动开启 iCloud 同步，所选数据会通过 Apple 的 iCloud / CloudKit 体系在您自己的设备之间同步；API Key 不会写入普通同步包，而是依赖 iCloud 钥匙串（如果系统与账户允许）。
  • 本地网络与调试：本地网络权限仅用于可选的局域网调试 / 远程文件访问功能。如果您主动连接自己的电脑调试工具，相关应用文件或请求流量可能会发送到您自己控制的本地调试端。

5. 反馈助手

仅当您主动使用反馈助手并提交反馈时，应用才会向反馈服务发送数据。

  • 用户主动填写的内容：反馈类型、标题、详细描述、复现步骤、预期行为、实际行为和补充说明。
  • 环境快照：平台、App 版本号、Build 号、操作系统版本、设备型号标识、语言地区、时区。
  • 极简诊断信息：时间戳、当前提供商数量、非临时会话数量、平台信息和 App 版本信息。
  • 防滥用校验：反馈请求会附带 challenge、时间戳、签名和 PoW 相关字段，用于限制垃圾提交。
  • 状态刷新：如果您之后在反馈中心刷新工单状态，应用会使用本地保存的工单编号和票据令牌向反馈服务查询进度。

反馈文本在提交前会做一层基础脱敏，尝试遮盖常见 Bearer Token、API Key 和部分密钥格式；但这并不是完整的数据防泄漏系统。请不要主动在反馈内容中填写密钥、身份证号、住址、健康信息等敏感内容。

6. 第三方服务、分析与开发者服务器边界

  • 第三方 AI 提供商：负责处理您主动发起的聊天、图像、音频等模型请求。
  • Apple 系统服务：如 WatchConnectivity、iCloud / CloudKit、iCloud 钥匙串等，由 Apple 依其自身政策处理。
  • 开发者运营的反馈服务：仅在您主动提交或查询反馈工单时使用。
  • 不自动上传的内容：聊天内容、本地记忆、本地请求日志、年度汇总数据和 API Key 不会默认自动上传到开发者控制的服务器。
  • 广告与通用埋点：当前代码中未集成第三方广告 SDK，也未发现通用型开发者侧行为分析 SDK。若未来加入此类能力，我们会先更新隐私政策。
  • Apple 分析共享：如果您在系统层面选择与开发者共享分析，Apple 可能会按照其规则收集匿名崩溃日志或诊断信息。

7. 免责声明

  • 中立工具属性：本应用是通用 LLM 客户端工具，可能预置部分服务端点或反馈服务地址，但默认不内置可直接使用的商业模型凭据。
  • 内容责任：所有 AI 生成内容均由您自行配置的第三方服务提供商返回，我们无法保证其准确性、合法性或适用性。
  • 服务选择：您选择使用的任何 API 服务、同步能力或反馈服务，都应由您自行评估其隐私政策、条款、成本与风险。
