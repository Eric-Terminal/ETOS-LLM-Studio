# 入门总览

ETOS LLM Studio 是一款在 iPhone 和 Apple Watch 上跑的 AI 客户端，你**自己带 API Key**接入 OpenAI、Anthropic、Google 或任何 OpenAI 兼容服务。App 本身不卖模型订阅、不收手续费、所有数据存在本机。

如果你**完全没用过**，请把这一页当作目录页，按顺序点开三篇核心入门教程，每篇大约 5–10 分钟。

## 第一次使用的最短路径

只做三件事就能用起来：

1. **装好 App** → [安装与首次启动](/guide/installation)（含 TestFlight、Xcode 编译两种方式；以及第一次启动后的权限设置）
2. **接入一个 LLM 提供商** → [第一次配置提供商](/guide/first-provider)（10 步流程，含 OpenAI / Claude / Gemini Base URL 速查表 + 错误码对照）
3. **发出第一条消息** → [第一次发起聊天](/guide/first-chat)（认识输入框周围的所有按钮，附件、工具开关、思考过程、导出怎么用）

走完这三步，你就拥有一个能正常聊天、能发图片、能朗读回答的 AI 助手了。**先停在这里**，不要急着开高级功能。

## 你会陆续接触到的概念清单

下面这些名词在 App 和文档里反复出现。**第一次见不需要全懂**，跟着教程走会被反复解释。这里给你一个心理预期：

| 名词 | 一句话说明 | 在哪用到 |
| --- | --- | --- |
| **提供商**（Provider） | 你接入的 LLM 服务商，比如 OpenAI、Anthropic | 设置 → 提供商与模型管理 |
| **模型** | 提供商下的具体 AI，比如 GPT-4o、Claude 3.5 Sonnet | 同上 |
| **会话**（Session） | 一组连续聊天消息的集合 | 聊天 Tab |
| **当前模型** | 当前发消息时用的那个模型 | 设置 → 当前模型 |
| **多模态** | AI 能看图、听声音 | 聊天页 `+` 附件菜单 |
| **MCP**（Model Context Protocol） | 给模型挂第三方工具的开放协议，像"模型用的插件市场" | 设置 → MCP 工具集成 |
| **Skills** | 一组本地脚本/资源，让模型有"专项技能" | 设置 → Agent Skills |
| **快捷指令工具** | 把 iOS Shortcuts 暴露给模型当工具用 | 设置 → 快捷指令工具集成 |
| **记忆** | 跨会话保留的事实，下次聊天会被自动带进去 | 设置 → 记忆系统 |
| **世界书** | 一份"按关键词触发注入"的知识库 | 设置 → 世界书 |
| **每日脉冲**（Daily Pulse） | 每天定时主动生成的"今日值得看什么"卡片 | 设置 → 每日脉冲 |
| **同步与备份** | iPhone ↔ Watch 数据互通，以及 ETOS 数据包导入导出 | 设置 → 同步与备份 |

::: tip 不要一上来全开
看到这么多功能很容易想一次性全开。**建议先把基础聊天稳定 2-3 天**，熟悉了再开记忆、世界书、Daily Pulse、MCP——这些功能彼此独立，开早开晚不影响。
:::

## 我现在还不该做什么

刚装好的前几天**不建议**做这些事：

- 一上来导入 1000+ 条历史会话（先确认你想用，导入大数据集会增加首启动时间）
- 同时启用 5 个以上 MCP 服务器（增加请求体积、容易触发模型 token 限制）
- 在 Apple Watch 上一项项配置提供商（屏幕太小，效率极低；先用 iPhone 配好再同步过去）
- 跑通之前就把 temperature、top_p、max_tokens 都改了（基础链路出问题时会很难判断到底是参数问题还是接入问题）
