# Daily Pulse

Daily Pulse 是 ETOS LLM Studio 里最不像“传统聊天客户端”的能力之一。  
它不是等你问一句答一句，而是主动生成一组“今天可能值得看什么”的情报卡片。

如果你想看它的真实输入源、默认阈值、卡片筛选逻辑和提示词骨架，请直接读 [Daily Pulse 设计原理](/design/daily-pulse)。

## 它解决的是什么问题

很多时候你并不是不知道怎么问，而是不知道今天该问什么。  
Daily Pulse 把这种“主动整理入口”的工作前置了。

## 主要能力

### 每日生成一组卡片

系统会根据当前配置和长期反馈，生成一组主动情报。

### 卡片可转任务

如果某张卡片值得后续跟进，可以直接转成待处理任务。  
这些未完成项会跨天保留，并参与下一次生成。

### 反馈会被学习

这些反馈不是一次性的：

- 点赞
- 降权
- 隐藏
- 保存

它们会逐步沉淀为长期偏好信号，影响后续结果。

### 提醒与继续聊

Daily Pulse 不只停留在“生成卡片”这一步，还包括：

- 晨间提醒
- 通知快捷动作
- 保存为会话
- 继续聊天

## 适合谁用

- 你希望 App 主动给你整理值得看的主题
- 你经常在固定时间浏览信息
- 你不想每次都从空白输入框开始

## 使用建议

### 模型选择

如果条件允许，给 Daily Pulse 单独选择一个适合整理与总结的模型，而不是和日常聊天完全共用。

### 反馈纪律

认真使用“点赞 / 降权 / 隐藏 / 保存”会比频繁换模型更能提升后续体验。

### 平台分工

- iPhone 端更适合完整浏览、保存和管理
- Apple Watch 端更适合快速看提醒、点开继续聊
