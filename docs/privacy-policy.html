隐私政策

最后更新：2025年12月27日

数据收集

ETOS LLM Studio 不会收集、存储或传输任何个人数据到我们的服务器。所有数据均存储在您的设备本地。我们尊重并保护您的隐私权。

本地数据存储

应用在您的设备上存储以下数据：

  • 聊天记录 - 您与 AI 的对话历史，完全存储在设备本地沙盒中
  • API 配置 - 您添加的 API 提供商和密钥信息，存储在应用沙盒中并受iOS系统文件保护机制保护
  • 记忆数据 - RAG 系统的向量化记忆，使用设备端向量化技术
  • 应用设置 - 您的偏好设置和自定义配置
  • 附件文件 - 您发送的图片和音频附件

这些数据仅存储在您的设备上，应用开发者无法访问。

第三方服务

当您使用 AI 聊天功能时，您的消息将发送到您配置的 API 提供商（如 OpenAI、Anthropic、Google 等）。这些服务商的隐私政策适用于他们处理您数据的方式。

重要提示：

  • 您可以自由选择任何兼容的 API 提供商
  • 所有网络通信均通过 HTTPS 加密传输
  • 我们不会拦截或记录您与第三方服务的通信内容
  • 请仔细阅读您所选择的 API 提供商的隐私政策

照片库访问

当您选择发送图片给 AI 模型时，应用会请求访问您的照片库。这些图片：

  • 仅用于发送给您配置的 API 提供商进行图像识别
  • 不会被上传到我们的服务器
  • 您可以随时在系统设置中撤销照片库访问权限

麦克风访问

应用请求麦克风权限用于语音输入功能。录制的音频：

  • 可以发送给支持语音识别的 API 提供商
  • 可以保存在设备本地作为对话附件
  • 不会在未经您同意的情况下发送到任何服务器

本地网络访问

应用包含"局域网调试"功能，允许您通过本地网络访问应用沙盒目录。此功能：

  • 需要您手动启动，默认关闭
  • 使用随机生成的6位PIN码保护
  • 仅在同一局域网内可访问
  • 严格限制在应用沙盒Documents目录内
  • 主要用于开发者调试和高级用户的文件管理

iCloud 同步

如果您启用了设备同步功能，部分数据（如会话列表、提供商配置）可能通过 iCloud 在您的设备之间同步。此时：

  • 数据同步遵循 Apple iCloud 的端到端加密机制
  • Apple 的 iCloud 隐私政策适用于此类数据传输
  • 您可以在应用设置中关闭同步功能

数据安全

我们采取以下措施保护您的数据安全：

  • 所有网络通信使用 HTTPS/TLS 加密
  • 应用使用标准加密技术，无需特殊出口许可
  • 应用数据存储在iOS沙盒中，受系统级访问控制保护
  • 应用遵循 Apple 的沙盒安全机制，其他应用无法访问您的数据

儿童隐私

ETOS LLM Studio 不针对13岁以下儿童设计，我们不会故意收集儿童的个人信息。

隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在应用内和本网页上发布，并标注更新日期。重大变更会通过应用内通知告知用户。
