# ETOS LLM Studio 隐私政策

**生效日期：** 2026年1月23日

## 1. 前言

ETOS LLM Studio 非常重视您的隐私。本应用（iOS 及 watchOS 版）的设计核心是保护用户数据。

**ETOS LLM Studio 的核心原则是：您拥有并控制您的核心数据。** 我们绝不存储您的聊天记录、语音录制或个人身份信息。这些敏感数据保留在您的设备上，且仅发送到您明确配置的 AI 提供商。我们可能运行有限的基础设施服务用于配置同步或统计活跃用户数量，但绝不触碰您的对话内容。

## 2. 数据收集与使用

### 2.1. 用户内容（聊天、音频、图片）
-   **本地存储：** 您的聊天记录、会话配置和记忆碎片使用加密的系统存储（沙盒）保存在您的设备本地。
-   **AI 处理：** 您发送的文本消息、录音和图片直接从您的设备传输到您配置的第三方 AI 提供商（例如 OpenAI、Anthropic 或自建的本地 LLM）。
-   **无中间商：** 我们不会拦截、存储或利用您的对话数据进行训练。

### 2.2. 语音数据
-   **麦克风使用：** 应用仅在您主动点击录制按钮发送语音消息时访问麦克风。
-   **处理：** 音频数据将作为附件发送，或通过您配置的 API 提供商转换为文本。语音数据不会发送给苹果的 Siri 或听写服务器。

### 2.3. 图片数据
-   **照片库：** 应用仅在您选择将图片附加到消息时访问您的照片库。我们不会扫描您的整个媒体库。

### 2.4. 同步机制
-   **设备间同步：** iPhone 与 Apple Watch 之间的数据同步使用苹果的 **WatchConnectivity** 框架。这是一种直接且加密的连接（通过蓝牙/Wi-Fi），不经过我们的服务器或 iCloud。

### 2.5. 本地网络
-   **远程文件访问服务：** 应用可能会请求访问本地网络的权限。这仅用于可选的“远程文件访问”功能，允许您通过在本地网络运行的专用工具（Python 程序）与应用进行交互。

## 3. 第三方服务

本应用仅作为客户端。您需对自己配置的 API 密钥和端点负责。请审阅您所使用的 AI 提供商（如 OpenAI、Anthropic、Cloudflare）的隐私政策，以了解数据离开本应用后的处理方式。

## 4. 分析与统计
我们可能会收集极少量的**匿名**遥测数据（例如：每日活跃用户数、应用版本号），以帮助我们评估项目健康度并优化开发。这些数据不包含任何个人身份信息、设备唯一标识符（UDID）或聊天内容。此外，如果您在 iOS 隐私设置中选择与应用开发者共享分析，苹果可能会收集匿名的崩溃日志。

## 5. 免责声明

本应用仅作为通用的 LLM 客户端工具。

1.  **中立工具属性：** 应用可能为了配置方便预设部分商业 API 服务端点，但默认**不包含**任何有效的 API 密钥或访问凭证。
2.  **内容责任：** 所有 AI 生成的内容均由您自行配置的第三方服务提供商生成。我们无法控制、干预或审查其内容，因此不对生成内容的准确性、合法性或合规性承担任何责任。
3.  **服务选择：** 您选择使用的任何 API 服务（包括我们可能在文档中推荐的服务）均属于您的个人选择。您需自行承担使用这些服务产生的费用及风险。
